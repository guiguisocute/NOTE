### 高级语言的“不确定性”格式

**确定性的语句**：
```c
int x = 114514;
int y = 1919810;
```
**不确定的语句**
```c
if(x >810){
    x++;
}
printf("%d",x);
```

#### `if`的杂项
- 省略大括号时，**只执行括号后下一条语句**，执行完后自动跳出，就只执行一条哈！。
- 如果`if();`————→直接忽略判断，执行后面的语句。
- 如果`if(){};`————→直接把条件判断和大括号里面全忽略了，执行后面的语句。(以上两条至少在GCC14.0的编译器里是这样的……)
- 
  ```c
  if(x >= 5){
    printf("%d",x)
  }else{
    printf("sb");
  }
  ```
    和下列代码一样吗？
    ```c
    if(x >= 5){
        printf("%d",x)
    }
    if(x < 5){
        printf("sb");
    }
    ```
    <font size=5%>不一样！</font>
    **你把大括号去了，语句再多加点else之类的就老实了**

`else`: 总之就是一定要跟着`if`的 **末尾**
